# Dashboard Demo

## GROMACS Trajectory Visualizer Dashboard

The screenshot below shows the complete dashboard analyzing a GROMACS trajectory:

![GROMACS Trajectory Visualizer Dashboard](dashboard-demo.png)

### Key Features Visible:

1. **Sidebar Controls** (Left panel)
   - File upload for `.xvg` files
   - Chart type selector (Line, Scatter, Area)
   - Data point markers toggle
   - Downsampling slider for large datasets

2. **Metric Cards** (Top row)
   - Total Frames: 16
   - Time Range: 0 – 150 ps
   - Mean, Min, Max values for selected columns

3. **Interactive Chart** (Center)
   - "Potential Stabilaity Over Time" visualization
   - Multiple data series plotted
   - Hover tooltips with exact values
   - Dark theme optimized for scientific visualization

4. **Raw Data Table** (Bottom left)
   - Full dataset view with scientific notation
   - Sortable and scrollable

5. **Descriptive Statistics** (Bottom right)
   - Statistical summary (count, mean, std, min, quartiles, max)
   - Quick insights at a glance

### Workflow:
1. Upload a GROMACS `.xvg` file
2. Customize chart options
3. Explore data interactively
4. Export as CSV for further analysis
